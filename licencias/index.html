1. Aplicación Rutas Irapuato

Rutas Irapuato es una aplicación móvil desarrollada por Brandon Kevin Ramirez Rosales para consultar rutas de transporte público, referencias y mapas de Irapuato, Guanajuato.

El nombre “Rutas Irapuato”, el icono, el diseño visual propio, la interfaz, la experiencia de usuario, el código de la aplicación, la estructura de presentación y el contenido original creado específicamente para la app pertenecen a su responsable, salvo que se indique lo contrario.

La información relacionada con rutas de transporte público, calles, colonias, referencias, puntos de interés, datos cartográficos y ubicaciones puede provenir de fuentes públicas, datos abiertos, observación propia, referencias comunitarias, fuentes oficiales o servicios de terceros. Dicha información se incluye con fines informativos y no implica titularidad exclusiva por parte de Rutas Irapuato.

Las rutas, recorridos, trazos, paradas o información operativa del transporte público pueden corresponder a autoridades, concesionarios, operadores de transporte u otras fuentes públicas correspondientes. Rutas Irapuato no representa necesariamente a dichas entidades ni garantiza que la información sea oficial, completa o actualizada en tiempo real.

2. Software y librerías de código abierto

Rutas Irapuato utiliza software de código abierto y paquetes de terceros. Cada componente conserva su propia licencia.

Flutter y Dart

La aplicación está construida con Flutter y Dart, tecnologías de Google distribuidas como software de código abierto.

Sitios oficiales: https://flutter.dev/ y https://dart.dev/

Paquetes Flutter/Dart

La app puede incluir, entre otros, los siguientes paquetes:

  • cupertino_icons
  • http
  • maplibre_gl
  • path_provider
  • geolocator
  • shared_preferences

Las licencias completas de estos paquetes se pueden consultar en sus repositorios, en pub.dev y dentro de los avisos de licencias que muestra la aplicación.

MapLibre

Rutas Irapuato utiliza MapLibre para mostrar mapas dentro de la aplicación.

MapLibre y sus componentes mantienen sus propias licencias de código abierto. La información de licencia del paquete Flutter puede consultarse en: https://pub.dev/packages/maplibre_gl/license

3. Mapas, datos cartográficos y atribuciones

OpenStreetMap

Parte de los datos cartográficos utilizados por la app o por proveedores de mapas puede provenir de OpenStreetMap y sus colaboradores.

OpenStreetMap es open data bajo la licencia Open Data Commons Open Database License (ODbL). OpenStreetMap requiere atribución a OpenStreetMap y sus colaboradores, así como referencia a la licencia aplicable.

Más información: https://www.openstreetmap.org/copyright

OpenFreeMap

Rutas Irapuato puede usar estilos y tiles de OpenFreeMap para los mapas de calles y mapas descargables.

El uso de OpenFreeMap está sujeto a sus propios términos de servicio y políticas.

Términos: https://openfreemap.org/tos/

Esri / ArcGIS World Imagery

La vista satelital de Rutas Irapuato puede utilizar imágenes y tiles de Esri / ArcGIS World Imagery.

La atribución de World Imagery puede incluir fuentes como Esri, Maxar, Earthstar Geographics y la comunidad GIS, según disponibilidad y zona consultada.

Información de atribución: Esri - cita de basemaps ArcGIS Online
