2. Datos personales que trata la aplicación

Rutas Irapuato no requiere crear una cuenta para consultar rutas, buscar referencias, planear viajes o usar el catálogo local.

La aplicación puede acceder o tratar los siguientes datos:

  • Ubicación aproximada o precisa del dispositivo: solo cuando el usuario otorga permiso desde Android o iOS y utiliza funciones como “Usar mi ubicación”, seleccionar origen con ubicación actual, ver rutas cercanas o planear un viaje.
  • Puntos de origen y destino: cuando el usuario selecciona puntos en el mapa, busca una referencia o planea un recorrido. En la versión actual, estos datos se procesan dentro del dispositivo para calcular sugerencias de ruta.
  • Preferencias guardadas localmente: la app puede guardar rutas favoritas, rutas recientes, estilo de mapa seleccionado, catálogo local y mapas descargados para uso offline.
  • Datos técnicos de servicios de mapas: cuando el usuario usa mapas en línea, vista satelital o descarga mapas, proveedores externos pueden recibir solicitudes técnicas necesarias para entregar el servicio, como dirección IP, tipo de dispositivo, zona del mapa solicitada, fecha y hora de la solicitud.
  • Datos de soporte: si el usuario contacta a soporte, podremos tratar los datos que proporcione voluntariamente, como nombre, correo electrónico, descripción del problema, capturas de pantalla o información técnica necesaria para atender el reporte.

Rutas Irapuato no solicita datos financieros, identificación oficial, contactos, cámara, micrófono, fotos, mensajes, calendario ni datos sensibles.

3. Finalidades del tratamiento

Los datos anteriores se utilizan para:

  • Mostrar la ubicación actual del usuario cuando este lo autoriza.
  • Permitir seleccionar origen y destino para planear viajes.
  • Consultar rutas de transporte público, referencias y mapas de Irapuato.
  • Descargar y conservar mapas para uso offline.
  • Recordar preferencias locales como favoritos, recientes y mapa seleccionado.
  • Proporcionar soporte técnico cuando el usuario lo solicite.
  • Mantener el correcto funcionamiento de la aplicación.

Rutas Irapuato no usa datos personales para publicidad personalizada, venta de datos, perfiles comerciales ni seguimiento entre apps o sitios de terceros.

4. Uso de ubicación

La ubicación se solicita únicamente cuando el usuario decide usar una función que la requiere. El usuario puede negar o revocar el permiso desde la configuración de su dispositivo.

Rutas Irapuato no accede a la ubicación en segundo plano. Si el usuario no otorga permiso, podrá seguir usando la app seleccionando manualmente origen y destino en el mapa o mediante búsqueda.

5. Mapas, proveedores externos y servicios de terceros

Rutas Irapuato puede utilizar servicios de terceros para mostrar mapas, vista satelital o descargar información cartográfica. Dependiendo del mapa seleccionado, pueden participar proveedores como OpenFreeMap, Esri / ArcGIS, MapLibre y OpenStreetMap.

Estos proveedores pueden procesar información técnica necesaria para entregar mapas o imágenes, como dirección IP, solicitudes de mapa, fecha y hora, zona consultada y datos técnicos del dispositivo o conexión.

El uso de estos servicios también puede estar sujeto a sus propias políticas de privacidad y términos. Rutas Irapuato no controla directamente las prácticas de privacidad de dichos proveedores.

6. Funcionamiento offline y almacenamiento local

La aplicación puede guardar datos en el dispositivo para permitir funcionamiento offline, incluyendo catálogo de rutas, mapas descargados, preferencias, rutas favoritas y rutas recientes.

Estos datos permanecen en el dispositivo del usuario y pueden eliminarse al borrar los datos de la app, desinstalar la aplicación o mediante opciones que se incorporen dentro de la app.

7. Transferencias y compartición de datos

Rutas Irapuato no vende datos personales.

Podemos compartir o permitir acceso a información únicamente en estos casos:

  • Con proveedores técnicos necesarios para operar mapas, descargas o soporte.
  • Cuando sea requerido por autoridad competente conforme a la ley aplicable.
  • Cuando sea necesario para proteger derechos, seguridad o funcionamiento de la app.
  • Con plataformas como Google Play o Apple App Store, conforme a sus propias políticas.

8. Conservación de datos

Los datos guardados localmente permanecen en el dispositivo hasta que el usuario los elimine, borre los datos de la aplicación o desinstale la app.

Los mensajes enviados a soporte se conservarán durante el tiempo necesario para atender el reporte y dar seguimiento razonable, salvo que la ley requiera conservarlos por más tiempo.

Los proveedores externos pueden conservar información técnica conforme a sus propias políticas.

9. Seguridad

Aplicamos medidas razonables para limitar la recopilación de datos, usar permisos del sistema operativo, almacenar preferencias principalmente de forma local y evitar solicitar información innecesaria.

Cuando se utilicen servicios remotos propios o de terceros, se procurará el uso de conexiones seguras mediante HTTPS.
